## Course Overview
CS405 Deep Learning introduces the fundamental principles, architectures, and practical methodologies of deep neural networks. The course emphasizes supervised deep learning with reasonable coverage of unsupervised and advanced models, combining theory with hands-on implementation using Python.

---

## Topics Covered
- Introduction to Deep Learning  
- Artificial Neural Networks (Perceptron, MLP)  
- Backpropagation, Optimization, and Training Strategies  
- Hyperparameter Tuning and Regularization  
- Convolutional Neural Networks (CNNs)  
  - Object Detection  
  - Semantic Segmentation  
- Deep Learning for Natural Language Processing (NLP)  
  - Word Embeddings, Word2Vec  
  - RNN, GRU, LSTM  
  - Seq2Seq Models and Attention  
- Transformers and Self-Attention  
- Generative Models (Autoencoders, GANs)  
- Introduction to Reinforcement Learning (Q-Learning)  
- Advanced Topics in Deep Learning  

Lab work includes Python basics, neural network implementation, CNNs, transfer learning, NLP preprocessing, transformers, and open-ended/project-based experiments.
